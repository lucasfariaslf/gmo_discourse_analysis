Nova versão Solicitação de Reconhecimento RTRS para Organismos de CertificaçãoEste documento, criado para a solicitação de reconhecimento formal de organismos de certificação sob padrões RTRS, foi modificado em conteúdo e formato.As modificações no tocante à versão anterior incluem:-Modificações de formato, numeração e nomes dos documentos.-Eliminação do ponto 6.3, sobre o procedimento para o cálculo de dias para Grupo e Multi-site (já está determinado no padrão de grupo e multi-site agora).-Eliminação do ponto 8.3, sobre revisores para o padrão de Cadeia de Custódia.-Eliminação do Anexo 4, sobre qualificações de revisores do padrão de Cadeia de Custódia.-É especificada a informação solicitada a respeito de esquemas similares de certificação, dentro das qualificações para Auditores para Cadeia de Custódia (Anexo 3).Além disso, é incluída uma quantia para o caso de ter que realizar revisões adicionais.Este formulário é utilizado tanto para o Padrão de Produção de Soja Responsável RTRS quanto para o Padrão de Cadeia de Custódia RTRS, de acordo com o estabelecido em seu ponto I.Solicitação: Solicitação para o Reconhecimento Formal de Organismos de Certificação para Padrões RTRS
